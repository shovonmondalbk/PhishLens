# PhishLens
PhishLens is a multimodal, explainable AI framework for real-time phishing website detection. The system combines three specialized analyzers - URL (DeBERTa), HTML (ELECTRA), and Visual (CLIP + OCR) - through a consensus-based Fusion Engine to achieve 99.40% accuracy and 99.32% F1 on a live 20,000-URL benchmark.
